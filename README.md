AI-Project-1-Rule-Based-Chatbot

A simple Rule-Based AI Chatbot developed using Python as part of an Artificial Intelligence Internship Project.

Features
* Handles greetings and common queries
* Uses rule-based responses
* Runs in a continuous loop
* Supports exit commands
* Uses Python dictionaries and conditional logic

Technologies Used
* Python 3
* Dictionaries
* Loops
* Conditional Statements

How to Run
python chatbot.py
